# 量化交易策略方案

## 一、策略概述

### 1.1 RSI 超买超卖策略

**原理**：
- RSI (Relative Strength Index) 相对强弱指标
- RSI < 30 → 超卖 → 买入信号
- RSI > 70 → 超买 → 卖出信号

**参数**：
- 周期：14日
- 超卖阈值：30
- 超买阈值：70

**优点**：
- 简单易懂
- 适用于震荡市场
- 信号明确

**缺点**：
- 单边行情中可能失效
- 需要结合趋势过滤

## 五、风控规则

1. **仓位管理**
   - 单只股票仓位 ≤ 95% 总资金
   - 预留5%缓冲应对手续费

2. **止损规则**
   - 单一策略最大亏损 10% 触发熔断

3. **审核机制**
   - 模拟交易收益率 ≥ 3% 方可申请实盘
   - 交易次数 ≥ 3笔

4. **实盘门槛**
   - 需要通过至少30天模拟交易验证
   - 收益率稳定 ≥ 5%
